# 60-Second Quickstart Checklist (v1)

Use this as a one-screen setup checklist.

## Install
- [ ] `pip install physical-mcp`
- [ ] Run `physical-mcp`

## Verify
- [ ] `physical-mcp --version` works
- [ ] `physical-mcp doctor` passes basic checks
- [ ] AI app shows camera tools (e.g. `capture_frame`)

## First success
- [ ] Capture one frame
- [ ] Ask for scene summary
- [ ] Create one watch rule
- [ ] Trigger one alert/event

## If blocked
- [ ] Re-run `physical-mcp`
- [ ] Check app-specific quickstart in `docs/`
- [ ] Check `docs/common-install-failures-v1.md`
- [ ] Open issue with logs + environment details
