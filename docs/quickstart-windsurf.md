# Quickstart: Windsurf + physical-mcp

## Prerequisites
- Windsurf installed
- Python 3.10+
- USB/UVC camera connected

## Install
```bash
pip install physical-mcp
physical-mcp
```
Setup attempts auto-configuration for supported app configs.

## Verify
1. Restart Windsurf.
2. Ask assistant to list connected MCP tools.
3. Confirm camera tools are available.

## First use
Try:
- `Capture a frame and summarize the scene.`
- `Create a watch rule for door/package events.`

## Troubleshooting
- Re-run `physical-mcp` after app updates.
- If local endpoint fails, run `physical-mcp doctor`.
